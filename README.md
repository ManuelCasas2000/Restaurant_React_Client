# Proyecto: Página web para restaurante
# Autor: Manuel Casas

# Instalación
yarn install

# Ejecutar
yarn dev

## Lenguaje y paquetes usados
Para este desarrollo se ha utilizado el lenguaje:
- React vite 4.1.0
Y los módulos:
- React Query
- Bootstrap
- Animate.css
- React Router Dom
- Redux toolkit
- React Redux
- Cloudinary (Se usará para cargar las imágenes)
- React Big Calendar + date fns + react datepicker
- React modal
- Sweet Alert 2
- APIRest
